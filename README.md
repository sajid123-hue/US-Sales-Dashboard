# 🏆 Adidas US Sales Dashboard (Power BI)

## 📌 Project Overview

This project focuses on analyzing the **Adidas US Sales dataset** using **Power BI** to transform raw transactional data into meaningful business insights. The dashboard provides a comprehensive view of sales performance, product trends, and regional profitability, enabling data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance using key KPIs:

  * Total Sales Revenue
  * Units Sold
  * Average Price per Unit
  * Operating Margin
* Compare **product category performance** (Men’s, Women’s, Footwear)
* Evaluate **regional profitability and efficiency**
* Analyze **sales channels** (Online, In-store, Outlet)
* Identify **top-performing states and markets**

---

## 🛠️ Tech Stack

* **Tool:** Power BI
* **Data Processing:** Power Query
* **Visualization:** Power BI Dashboard
* **Dataset Source:** [Adidas US Sales Dataset](https://data.world/stellabigail/adidas-us-sales-datasets)

---

## 📊 Key Insights

### 1. Channel Performance

* **Online sales** lead in volume (~0.22M units)
* **In-store sales** generate the highest revenue (~39.63%)

### 2. Regional Analysis

* **Top regions:** Northeast and West (highest margins)
* **Lowest performance:** Southeast region (needs optimization)

### 3. Product Trends

* **Top categories:**

  * Women’s Apparel (~83K units)
  * Men’s Apparel (~81K units)
* Apparel shows consistent demand compared to footwear

### 4. Geographic Insights

* Top-performing states:

  * New York ($64.23M)
  * California ($60.17M)
  * Florida ($59.28M)
* Significant opportunity in underperforming states
